I sorgenti Java includono le descrizioni. Ci vuole generare la documentazione di Javadoc.

Primo file: l'oggetto del gioco, formati di input e output.
Secondo file: il visualizzatore e un esempio di file che rispetta la sintassi chiesta per il file di output (input per il visualizzatore).

Per creare il file di output, lanciare il programma

    java Main/Main ../input.txt ../output.txt

nella cartella Motore/src e dove input.txt è il nome del file di input esistente e output.txt è il nome del file di output da scrivere.

Per visualizzare il gioco, lanciare il programma

    java AnimazioneOca ../../Motore/output.txt

nella cartella Animazione/src e dove output.txt è il nome del file di output generato prima.
